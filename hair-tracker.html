① Supabase 프로젝트 생성
supabase.com 무료 가입 → New Project 생성
② DB 테이블 생성
Supabase → SQL Editor → 아래 버튼 클릭 후 복사된 SQL을 붙여넣고 Run
③ Supabase API 키 확인
Project Settings → API → Project URL + anon public key 복사
④ Anthropic API Key (AI 사진 분석용)
console.anthropic.com → API Keys → Create Key

입력한 키는 이 브라우저의 localStorage에만 저장됩니다.
